tipo-cambio-peru

Tipo de cambio oficial del Perú (BCRP, SBS, SUNAT) en una sola llamada. Zero dependencies. Mantenido por Securex.

Ver en GitHub Instalación securex.pe

Por qué existe

Si construyes algo en Perú que toca dólares — una app contable, un dashboard de tesorería, un bot de Telegram, un sistema de conciliación bancaria — vas a necesitar el tipo de cambio oficial.

Hay tres fuentes:

AutoridadFunciónQuirk principal
BCRPTipo de cambio interbancario diarioAnexa HTML basura tras el JSON cierre
SBSTC oficial publicado por la regulación bancariaRota nombres de campos cada cierto tiempo
SUNATTC contable mensual usado para liquidacionesHTML con encoding latin1 / UTF-8 inconsistente

Cualquiera que haya hecho fintech en Perú escribió tres scrapers defensivos para esto. Esta lib hace la opción 3: validación local + abstracción de las quirks.

Instalación

npm install tipo-cambio-peru

Uso rápido

import { getTCToday, getBCRPRange, calcStats, convert } from 'tipo-cambio-peru';

// 1. TC de hoy (último día hábil)
const today = await getTCToday();
console.log(today);
// → { date: '2026-04-25', buy: 3.4592, sell: 3.4791 }

// 2. Histórico últimos 30 días
const range = await getBCRPRange('2026-03-26', '2026-04-26');
const stats = calcStats(range);
console.log(stats);
// → { days: 18, avg: 3.4315, volatilityPct: 1.07, ... }

// 3. Conversión USD → PEN
const soles = convert(1000, 'USD', 'PEN', today);
console.log(`USD 1000 = S/ ${soles}`);

El quirk más famoso del BCRP API

El BCRP devuelve JSON válido pero le pega HTML basura al final del payload (probablemente display_errors activado en su PHP). Un JSON.parse() ingenuo crashea con SyntaxError: Unexpected non-whitespace character after JSON.

El fix:

const text = await fetch(url).then(r => r.text());
const htmlStart = text.indexOf('<');
const jsonOnly = htmlStart > 0 ? text.slice(0, htmlStart).trim() : text.trim();
const data = JSON.parse(jsonOnly);

Strip todo desde el primer <, parse lo que quede.

API completa

FunciónDescripción
getTCToday()TC del último día hábil disponible (BCRP).
getBCRPRange(start, end)Histórico de TC interbancario entre fechas.
getBCRPSeries(name, start, end)Serie BCRP específica (compra, venta, bancario, etc.).
calcStats(rows)Promedio, min, max, std, volatilidad sobre serie.
convert(amount, from, to, rate)Conversión PEN ↔ USD con tasa dada.
